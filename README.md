# Plasma biomarker neuropathology analysis — code

R code that reproduces the tables and figures of the manuscript from a small synthetic demonstration dataset. The dataset was generated by `simulate_data.R` (no real participant data are used). 

License: MIT.

## System requirements

- R 4.3.3 
- R packages: tidyverse, pROC, ggbeeswarm, rstatix, ggpubr, patchwork, scales,  gt, lme4, lmerTest, emmeans, broom, broom.mixed, survival, survminer. 
- No standalone hardware required.

## Installation guide

1. Install R 4.3+ 
2. Install the packages

-Typical installation time: 5-10 minutes.


## Demo 

From the `code/` directory, build the synthetic dataset once (already pre-built), then run any table or figure script:

```bash
cd code
Rscript simulate_data.R          # writes ../demo_data/demo_cohort*.rds
Rscript Table1.R
Rscript Figure1.R
# ... Table2.R, Figure2.R, Figure3.R, Figure4.R,
#     ExtendedDataFigure1.R, ExtendedDataFigure2.R
```

`simulate_data.R` creates a `demo_data/` folder with the synthetic cohort. Each figure/table script reads it and writes its products (PDF figures, CSV/HTML tables) to a `demo_output/` folder. Expected outputs and run times:

| Script | Output in `demo_output/` | Run time |
|--------|--------------------------|----------|
| `simulate_data.R` | `demo_cohort.rds`, `demo_cohort_long.rds`, `demo_cohort.csv` | ~2 s |
| `Table1.R` | `Table1.html`, `Table1.csv` | ~4 s |
| `Table2.R` | `Table2.html`, `Table2.csv` | ~3 s |
| `Figure1.R` | `Figure1a`–`Figure1e` PDFs | ~6 s |
| `Figure2.R` | `Figure2a`–`Figure2d` PDFs | ~5 s |
| `Figure3.R` | `Figure3a`–`Figure3d` forest PDFs | ~3.5 min |
| `Figure4.R` | `Figure4a`–`Figure4d` PDFs | ~10 s |
| `ExtendedDataFigure1.R` | `ExtData1a`–`ExtData1c` PDFs | ~4 s |
| `ExtendedDataFigure2.R` | `ExtData2a`–`ExtData2c` PDFs | ~4 s |

# Instructions for use:

- To reproduce the manuscript tables and figures, run the above commands after replacing the synthetic dataset with the real participant data (not included here).
